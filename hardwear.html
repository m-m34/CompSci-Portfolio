What is a motherboard and what does it do?

A motherboard is a circuit board containing the principle components of a computer or other devices.

The motherboard allowes the computer to work.

What is CPU and what does it do?

CPU stands for central processing unit.

The CPU is like the brain of the computer where most calculatuions take place

The processor interprets and executes instructions and data contained in software programs.

What is RAM and what does it do?

RAM stands for Random Acces Memory.

The RAM is the computers memory.

RAM is volitile memory so it is basically only used when the computer is on.

What is a hard drive and what does it do?

The hard drive is the computer's main storage media device that permanently stores all data on the computer.

The hard drive stores the data.

What is a power supply and what does it do?

The power supply is the electricity the computer needs.

It supplies various components of the computer:

  1. The fans
  2. The CPU
  3. The RAM
  4. The hard drive

How do the differnt components communicate with one another?

Computer components are connected to each other via thin connections known as 'bus' on the motherboard.

Since multiple devices can be connected to the same bus, the communications are handled by the 'bus controller'.

Devices that can directly communicate with other devices without involving the main CPU and RAM are called 'bus mastering' devices.
